# WebLogic Simple Demo App

This repository contains a **simple Java web application** deployed on WebLogic Server.  

## What’s included
- `sampleapp.war` — WAR file of the application
- Screenshots of deployment 

## Steps I followed
1. Created a simple web project with basic HTML content.
2. Built a WAR file.
3. Started WebLogic Server (AdminServer).
4. Deployed the WAR file using WebLogic Admin Console.
5. Verified the application is running in browser.

## How to test
1. Download the WAR file.
2. Deploy it on your WebLogic Server via Admin Console.
3. Open `http://localhost:7001/sampleapp` in your browser.
